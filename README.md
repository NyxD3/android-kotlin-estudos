# Sistema de Biblioteca em Kotlin

Projeto desenvolvido em Kotlin com foco em prática de lógica de programação e orientação a objetos.

O sistema simula o funcionamento básico de uma biblioteca, permitindo cadastrar livros, realizar empréstimos e devoluções.

## Funcionalidades

- Cadastro de livros
- Listagem de livros
- Empréstimo e devolução
- Controle de disponibilidade

## Tecnologias

- Kotlin
- Programação Orientada a Objetos
- Git e GitHub

## Objetivo

Projeto criado para fins de estudo.
